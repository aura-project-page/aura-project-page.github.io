Consistent Multi-Subject Video Generation
via VLM-Grounded Semantic Alignment

Zixiang Zhou†, Zhentao Yu, Yifeng Ma, Hongmei Wang, Wenqing Yu, Cong Wang, Zilin Yang, Rui Chen, Jiarong Ou, Yezhou Liu, Yuan Zhou, Qinglin Lu
Tencent Hunyuan
†Corresponding author: zxzhou0916@163.com
Code 📄 arXiv:2607.04311 🤗 HuggingFace

We present Aura, a unified framework for high-fidelity, identity-consistent multi-subject video generation. Given heterogeneous reference images for people, objects and scenes together with a structured director-level prompt, Aura synthesizes a video that faithfully follows the script while preserving every subject’s identity and the surrounding scene context.

Examples
For each example below we show the reference image used as visual conditioning, the full structured prompt driving generation, and the resulting video. The reference image and prompt together define every subject (person, object, scene) along with the director-level shot script. Hover a video to reveal controls.
